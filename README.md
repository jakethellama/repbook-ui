# RepBook UI

### Overview

[RepBook](https://repbook.app) is a full-stack workout-tracking app. This UI is made with Angular and Tailwindcss. 

Also, I used AWS S3 and CloudFront to store and serve frontend content.


### Developing
```
npm run start
```

### Building
```
npm run build
```
